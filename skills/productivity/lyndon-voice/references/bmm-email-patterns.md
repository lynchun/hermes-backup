# BMM Email Patterns — Lyndon's Professional Voice

Extracted from real work emails in the BMM case (de-identified).

## Opener Patterns

```
Hi Guys,
Hi Eric,
Hi Tim/Katrina,
Hi Katrina,
```

Never: "Hey", "Dear", "Hello"

## Body Style

- Direct, no preamble. Gets to the point in first sentence.
- Lays out options clearly: "Our options for FX conversion are either done by you before remittance or done by us upon receipt."
- Polite but not deferential: "Can you please give us some detail/information regarding..."
- Uses "re" for regarding: "some details re the Diluca accounts"
- Precise with numbers: "EUR is rallying heavily against the AUD at 1.634 tonight"
- Measured, analytical tone. No emotional language. No exclamation marks.

## Sign-off

```
Regards,

Lyndon Arthurson
Adviser, Stewards Capital Group Pty Ltd
Mob +61 (0)479 083 743
[full signature block]
```

## Key Patterns

| Do | Don't |
|---|---|
| "Hi Guys," | "Hey guys" |
| "Can you please..." | "Can u pls" |
| "We are interested in" | "We wanna" |
| "Regards," | "Cheers" |
| Full signature | No signature |
| Concise, factual | Chatty, casual |

## Pronouns

- "We" = Stewards Capital / the firm
- "I" = personal position/view
- Consistent throughout — never mixes within same email
