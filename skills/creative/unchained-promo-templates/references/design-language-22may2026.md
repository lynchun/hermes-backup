# Unchained Promo Video Design Language (from Lyndon, 22 May 2026)

Captured verbatim from Lyndon's description of his existing iMovie promos. These are the production principles and visual language he developed over years of hands-on editing.

## The Fundamental Rule: Beat-Synced Editing

Every edit must land ON the kick drum or the snare. Never between. The choice of which drum depends on the emotional effect you want:

- **Kick drum** = weight, grounding, power — use for logo reveals, artist name drops, EP title reveals
- **Snare** = sharp, cutting, energy — use for tension releases, sudden reveals, scene transitions
- Alternating kick/snare = rhythmic flow — use for image slideshows that build anticipation

## Dynamic Pacing Mirrors the Track

The visual tempo must follow the music's energy curve exactly:

- **Low energy** (intro, breakdown, ambient section) → slow image movements, gentle fades, lingering shots, subtle/unobtrusive logo animation
- **Building tension** → gradually increase animation speed, add more frequent cuts, start zooming in
- **The Drop** → sudden switch to high energy: fast cuts, aggressive logo animation, beat-synced flash effects, rapid image transitions
- **Release/Hold** → pull back, slow down, let the reveal breathe and land

The core emotional theme across all videos: **anticipation and tension** — build it, sustain it, then release it at the drop.

## Logo Treatment

The Unchained label logo is a flexible design element that can appear:
- As a cold open (fades out before music)
- As a closing frame (final image)
- Bookending the video (both)
- Throughout the video at key moments
- Animated dynamically on the beat (scale pulse, rotation, position shift)

## Recognised Template Styles

### A: TV Test Pattern Intro
Opens with old-school broadcast test pattern. Artist name + catalogue number overlaid. Creates retro vibe and suspense before the drop. Raises brand profile through creative identity.

### B: Artist Portrait Split (Left/Right)
Artist face from cover art on the LEFT. Song/track name on the RIGHT. Edits on kick or snare. Clean, editorial feel. (CESCO, OAKK, GROVES releases)

### C: 3D Name Zoom (Fiverr Commission)
Artist name rendered in 3D — starts micro, slow zooms in as tension builds. Reaches full size at the drop revealing artist + EP title. Marvel movie intro style — cartoonish but dark. Lyndon is tired of this approach but it remains a valid template. Can still commission new 3D renders.

### D: Quick Cover Art Slide-Through
Rapid slideshow of cover art images, each hitting on a kick/snare. Builds anticipation. Culminates in artist + EP reveal. (Lost Sirens release)

### E: Dripping Numbers Logo Intro
Animated dripping/falling numbers around the Unchained logo as intro sequence. Creates suspense before the drop.

### F: Unchained 3D Animated Logo
Commissioned 3D animation using cover arts as dynamic flow within lettering. Marvel movie style.

## The Ultimate Goal

Build a template library in HyperFrames where each new release is:
1. Pick a template style
2. Supply cover art + audio + artist name + EP name + catalogue number
3. System generates beat-synced, dynamically-paced promo video
4. Review, tweak, render

No more iMovie manual editing.
