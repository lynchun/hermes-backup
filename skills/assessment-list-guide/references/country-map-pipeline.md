# GeoPandas Country Map Pipeline

Techniques learned from building 6 continent reference maps.

## Shapefile resolution
- **110m** (ne_110m_admin_0_countries): Fast, small. OK for large countries. Microstates missing (Liechtenstein, Monaco, Andorra, San Marino, Malta, Vatican). Chile's southern portion disconnected visually.
- **50m** (ne_50m_admin_0_countries): Better. Chile renders as 31 connected parts vs 2. All microstates present. Use by default.
- Download: `curl -L -o ne_50m.zip "https://naciscdn.org/naturalearth/50m/cultural/ne_50m_admin_0_countries.zip"`

## Country name matching
Natural Earth names differ from common names. Keep a mapping dict:
```python
NAME_FIX = {
    "Czech Republic": "Czechia", "United States": "United States of America",
    "Bahama Islands": "Bahamas", "Dominican Republic": "Dominican Rep.",
    "South Korea": "South Korea", "Congo": "Congo",  # Republic of, not DRC
}
```
Always check `world[world['NAME'].str.contains('X', case=False)]` before assuming a name.

## Mercator compensation (CRITICAL)
At latitudes away from the equator, WGS84 geographic coords produce horizontally stretched maps. Fix:
```python
mid_lat = np.radians((ymin + ymax) / 2)
aspect = 1.0 / np.cos(mid_lat)
ax.set_aspect(aspect)
```
Europe at 52N needs ~1.64 aspect. Asia at 25N needs ~1.10. Without this, maps look "squashed."

## Perfectly round circles
`mpatches.Circle()` stretches with `set_aspect()`. Use `ax.scatter()` instead — always perfectly circular:
```python
ax.scatter(cx, cy, s=scatter_size, c='white', edgecolors='none', zorder=10)
ax.annotate(str(i), (cx, cy), fontsize=6.5, fontweight='bold', ha='center', va='center')
```

## Circle placement
- Use `geom.representative_point()` — guaranteed inside polygon (unlike `centroid` which can be outside for irregular shapes)
- `centroid` for Chile in 110m data: falls in southern disconnected tip. `representative_point` in 50m: correct central position.

## Leader lines for tiny countries
Countries with area < 2.0 deg² at map scale need leader lines:
```python
area_deg2 = geom.area
is_tiny = area_deg2 < 2.0  # Qatar 1.0, Lebanon 1.0, Jamaica 1.1, Puerto Rico 0.8, Bahamas 1.4
```
Pattern: small dot on country + visible dark line (1.2pt, #444444) + numbered circle placed outside (offset = 4% of map width).

## Unshaded land visibility
The page background (#fdfaf3) is very light. Unshaded land must be clearly distinguishable:
```python
LAND_UNSHADED = '#e8e0d0'  # distinct from #fdfaf3 background
EDGE_ALL = '#b8af9a'       # visible borders between unshaded countries
```

## Greenland and political correctness
Greenland is geographically North America but politically Danish territory. User wanted it in Europe. Natural Earth classifies it as "North America" — we manually moved it. Handle political sensitivities on a case-by-case basis.

## Token-efficient map generation
All 5+ continent maps should be generated by ONE script in ONE run. Don't generate individually — too many LLM calls.
